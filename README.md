# 🧠 Emoji Memory Match

A fun, responsive memory matching game built with HTML, CSS, and JavaScript. Flip cards to find matching emoji pairs. Great for all ages!

---

## 📱 Preview

![Game Screenshot](preview.png)

---

## 🎮 How to Play

- Click or tap any two cards to flip them.
- Try to remember the emoji and find its matching pair.
- The game ends when all pairs are matched.
- Track your number of moves and how fast you complete it!
